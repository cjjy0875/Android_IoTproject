-주제선택
	실생활에 가까운 기술을 위주로 찾아 봤다.
	하차벨을 선택한 이유 우리가 수업시간 때 배운 기술을 가장 효과적으로 활용할 수있어서	

-사용하는 기술
	안드로이드 스튜디오 : 버튼을 누르면 AWS와 연동시킨다
	ESP LED : 하차벨을 누르면 불이 켜진다 (스피커도 회로에 추가해 소리도 넣어보자)
	ESP32 비콘 : 사용자가 안드로이드 스튜디오에서 보내는 메세지를 이용해 LED불을 조종
		-AWS SNS기술 이용
	
	PIR sensor : 버스안 인원 파악
		-AWS DB와 연동해 인원수 저장을 한다.

-업그레이드 가능한점
	추가로 버스안에 인원수를 Pir sensor을 이용해 체크할 수있게 구현하였다.
	이 기술은 영유아 스쿨버스나 학원 버스 등에서 유용하게 이용할거 같다(안내린 사람을 선생님이 체크 가능) 

	
-안드로이드 버전 관리
	java
	API 27 Android 8.1 (Oreo)
